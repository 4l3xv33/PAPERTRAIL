# PAPERTRAIL Roadmap

This is a living document for PAPERTRAIL.

## ✅ MVP

- Static GitHub Pages frontend.
- Cloudflare Worker proxy for OpenAlex.
- Researcher search by name, ORCID, or OpenAlex Author ID.
- Browser-local IndexedDB cache.
- Author profile and works analysis.
- Markdown memo generation.
- Markdown and JSON export.

## 🚧 Planned Features

- Configurable provider endpoint in the UI.
- More resilient OpenAlex pagination controls.
- Additional visual summaries for collaboration patterns.
- Import/export of browser-local cache snapshots.

## 💡 Future Ideas

- ORCID provider adapter.
- Crossref provider adapter.
- Semantic Scholar provider adapter.
- PubMed provider adapter.
- Saved local comparison workspaces.
